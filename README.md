# Project Name 
Personal Profile Card
# Description
A dark, terminal-style developer profile card made with plain HTML/CSS.
Has a cursor, status badge, bio, stats, tech tags, and hire/resume buttons.
No frameworks, fully responsive, easy to customize.
# Features
Terminal-style design with cursor and status badge
Bio, stats grid, and tech stack tags
Hire Me / Resume buttons with social links footer
Pure HTML/CSS, no frameworks
Fully responsive on mobile
# Technologies Used
- HTML5
- CSS3
